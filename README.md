# Mini CRM - Client Lead Management System

A futuristic SaaS-level CRM built with React and Node.js.

## Prerequisites
- Node.js (v16+)
- MongoDB (running locally)

## Getting Started

### Backend
```bash
cd server
npm install
node seed.js # Create initial admin
node server.js
```

### Frontend
```bash
cd client
npm install
npm run dev
```
